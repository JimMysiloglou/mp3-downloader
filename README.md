# Mp3 download automation with python
This is a simple python program downloading videos from a certain youtube list convert it to mp3 format, normalize the tracks and coping them to usb stick.

## Requirements
* Python >= 3.7
* Pillow >= 8.3.1
* psutil >= 5.8.0
* pydub >= 0.25.1
* pyudev >= 0.22.0
* tk >= 0.1.0
* youtube-dl >= 2021.6.6
